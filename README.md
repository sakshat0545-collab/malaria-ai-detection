# 🧬 Malaria AI Diagnostic System

An AI-powered system that detects malaria parasites from microscopic blood smear images using deep learning.

## 🚀 Features

- AI-based malaria parasite detection
- 94% model accuracy
- Real-time image analysis
- Confidence score display
- Professional diagnostic dashboard
- Image preview before analysis

## 🧠 Model

The system uses a **CNN with EfficientNet architecture** trained on malaria cell images.

Dataset Source:
NIH Malaria Dataset

## 📊 Performance

| Metric | Value |
|------|------|
| Training Accuracy | 94.6% |
| Validation Accuracy | 93.6% |

## 🖥️ Tech Stack

- Python
- TensorFlow / Keras
- Flask
- HTML / CSS
- Railway Deployment

## 📷 Application Workflow

1. Upload blood smear image
2. AI model analyzes cell structure
3. System predicts infection status
4. Confidence score displayed

## ⚠ Medical Disclaimer

This system is a **research prototype** and should not be used for clinical diagnosis without professional validation.
